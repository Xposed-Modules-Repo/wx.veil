# 模块简介
面纱，隐藏微信的聊天记录, 通讯录列表，拦截语音视频通话，防止消息撤回等。

支持8.0.58及以上的版本，GP版不做适配

测试的版本可在【模块设置】-【支持版本】下载




# 使用教程

使用前请在LSPosed Manager的设置界面关闭**Xposed API调用保护**(切换需要重启)

建议使用it版本的LSPosed

```plain
点击主界面右上角的搜索按钮，在搜索框输入 #veil 进入模块配置界面
在好友界面长按好友备注5秒或在聊天输入框输入 #veil 打开对该好友隐藏的界面
更多使用教程请在模块里面打开查看
```
